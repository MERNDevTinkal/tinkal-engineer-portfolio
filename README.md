## üöÄ Getting Started

1.  **Clone & Install:**
    ```bash
    git clone https://github.com/MERNDevTinkal/Portfolio-Website.git
    cd Portfolio-Website
    npm install
    ```

2.  **Environment Variables:**
    Create a `.env` file and add:
    ```env
    GROQ_API_KEY=your_groq_api_key
    NEXT_PUBLIC_EMAILJS_SERVICE_ID=your_id
    NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=your_id
    NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=your_key
    NEXT_PUBLIC_CONTACT_FORM_RECEIVER_EMAIL=your_email@example.com
    ```

3.  **Run Locally:**
    ```bash
    npm run dev
    ```
